# una-ihcux-lista01
Abra o seu CMD (ou PowerShell) e execute a seguinte sequência de comandos:

Vá para a sua Área de Trabalho (cd Desktop).
Crie uma pasta chamada missao-terminal.
Entre nela e crie um arquivo de texto vazio chamado sucesso.txt.
Dica de Sênior: No Windows, você pode usar o comando type nul > sucesso.txt ou echo pronto > sucesso.txt.

Use o comando dir para listar o conteúdo da pasta.
